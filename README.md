# E-Commerce Data Analysis

This project analyzes clothing sales data across California to uncover customer purchasing behavior, sales trends, product performance, and order fulfillment efficiency. The dataset contains transaction-level records, including product details, pricing, quantities sold, customer demographics, sales channels, customer ratings, geographic locations, order dates, shipping dates, and delivery times.

Using Python and data analysis libraries such as Pandas, NumPy, Matplotlib, and Seaborn, this project explores key business metrics, identifies sales patterns, evaluates channel performance, and generates actionable insights to support data-driven decision-making in the e-commerce industry.

## Key Areas of Analysis
- Sales Performance Analysis
- Product Performance Analysis
- Customer Demographics Analysis
- Sales Channel Comparison
- Geographic Sales Distribution
- Customer Rating Analysis
- Delivery and Fulfillment Performance
- Business Insights and Data Visualization

## Dataset Features
- Transaction ID
- Product Name
- Quantity Sold
- Unit Price
- Sales Amount
- Order Date
- Ship Date
- Customer Gender
- Order Mode
- Customer Rating
- State and County
- Delivery Time
